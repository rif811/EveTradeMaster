EVE Online: Industry & Trade Master (v13.0)

EVE Online: Industry & Trade Master — это мощный инструмент на Python для автоматизации торговых и производственных расчетов в мире EVE Online. Приложение использует официальный ESI API и промышленный движок Fuzzwork для предоставления актуальных данных в реальном времени.
основной функционал
1. Межхабовая торговля (Арбитраж)

    Сравнение цен продажи (Sell) в одной системе с ценами покупки (Buy) в другой.

    Автоматический расчет профита в процентах.

    Подробная индикация процесса скачивания страниц рынка.

    Удобство: Клик по любой строке в таблице копирует название товара в буфер обмена.

2. Лучшая цена продажи (Срез цен)

    Мгновенное получение ТОП-3 минимальных цен продажи на главных станциях 5 торговых хабов: Jita, Amarr, Dodixie, Rens, Hek.

    Автоматическая подсветка самой выгодной цены (зеленым) и самой дорогой (красным).

    Фильтрация только по основным станциям (Jita IV-4 и т.д.), исключая "мусорные" ордера в системе.

3. Чертежи (Промышленный BOM)

    Глубокая рекурсия: Полное раскрытие цепочки производства от Т2-кораблей до Т1-компонентов и базовых минералов.

    Складская логика (Surplus Stock): Уникальный алгоритм, который учитывает дискретность производства. Если реакция выдает 10,000 единиц Carbide, программа использует остатки для других узлов дерева, не допуская завышения количества ресурсов ("ошибка миллиардов").

    Точный Yield: Учет выхода продукции для реакций (10,000 шт) и топливных блоков (40 шт).

    Группировка: Автоматическое распределение базовых ресурсов по папкам: Минералы, Планетарка, Газы и т.д.

4. Глобальный поиск (High-Sec Scanner)

    Сканирование всех ордеров в 9 крупнейших регионах High-Sec (безопасность 0.5+).

    Сравнение найденных цен со всеми Buy-ордерами в 5 главных хабах.

    Расчет чистой прибыли в ISK с учетом игрового налога 4.2%.

    Интерактивный прогресс-бар выполнения задачи.

Технические особенности

    Персистентный кэш (Вечная база):

        id_map.json — кэш идентификаторов (Название -> ID).

        names_db.json — база названий всех когда-либо найденных предметов.

        blueprints_db/ — локальная папка с составами чертежей.

        Результат: При повторном запросе данные грузятся мгновенно без интернета.

    Активные статусы: Строки состояния выполнены в виде текстовых полей — любую ошибку или системное сообщение можно выделить и скопировать.

    Многопоточность: Все расчеты и запросы к API вынесены в отдельные потоки, благодаря чему интерфейс не зависает.

Установка и запуск

    Установите Python (3.10 или выше).

    Установите необходимые библиотеки:
    code Bash

    pip install customtkinter requests urllib3

    Запустите основной файл:
    code Bash

    python main.py

Компиляция в .exe

Для создания исполняемого файла используйте PyInstaller:
code Bash

pyinstaller --noconsole --onefile --collect-all customtkinter main.py

Важные примечания

    Сброс данных: Если расчеты в модуле "Чертежи" кажутся неверными (из-за старых версий программы), просто удалите папку blueprints_db и файл names_db.json. Программа создаст новую чистую базу с актуальной математикой.

    Копирование: Программа поддерживает горячие клавиши Ctrl+C вне зависимости от текущей раскладки клавиатуры.

    ESI API: Приложение работает с публичными данными и не требует ввода логина/пароля от игрового аккаунта.
